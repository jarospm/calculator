# Calculator

A simple on-screen calculator built as part of [The Odin Project](https://www.theodinproject.com/lessons/foundations-calculator) Foundations curriculum.

**[Live Demo](https://jarospm.github.io/calculator/)**

## About

Functional calculator supporting basic arithmetic, square root, decimal input, and keyboard controls. Handles edge cases like division by zero and chained operations.

## Technologies & Concepts

- HTML, CSS, JavaScript
- DOM manipulation and event handling
- State management for multi-step operations
- Keyboard input support
- Error handling
